# Health Check Reports

Эта папка содержит автоматически генерируемые отчёты о проверке репозитория.

Отчёты создаются:
- Ежедневно в 12:00 по Москве
- При каждом коммите в main
- Вручную через GitHub Actions

Формат: `health-check-YYYY-MM-DD.md`
